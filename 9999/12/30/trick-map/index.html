 一些经常用到却经常忘记的东西，避免总是搜

  • comm 求差集 交集

    comm set1.txt set2.txt、

    输出3列，每一列使用制表符\t隔开。

    第一列是set1.txt中有而set2.txt中没有的，第二列则是set2.txt中有而set1.txt中没有的，第三列是set1.txt和set2.txt中都有的。

  • Tar 解压

    gz格式 tar -xzvf xxx.tar.gz

    bz格式 tar -jxvf ×××.tar.bz2

  • Tar 压缩

    gz tar -czvf xxx.tar.gz

  • cut 取字段

    cut -f n

  • Paste 按列合并文件

    paste file1 file2

  • crontab 每小时运行

    0 */1 * * *

  • C++ long long 格式化

    %lld

  • screen 日志

    screen -L xxx

  • sort 分隔符

    sort -t $’\t’

  • python 列表倒序

    l[::-1]

  • python enumerate

    for i, element in enumerate(seq):

  • Nerdcommenter

    注释 \cc

    取消注释 \cu

  • Vim 跳转 前进 后退

    ctrl+i ctrl+o

  • crtl+s 怎么办

    ctrl+q

  • 开启core

    ulimit -c unlimited

  • sed删除所有空格

    sed 's/ //g'

  • awk 求和

    awk '{sum += $1};END {print sum}'
